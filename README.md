# داش كام السيارة للآيفون PWA

تطبيق ويب تقدمي يعمل من Safari على iPhone.

## المزايا
- كاميرا خلفية أو أمامية.
- تسجيل فيديو من داخل المتصفح.
- رسم التاريخ والوقت والسرعة على الفيديو نفسه عبر Canvas.
- سرعة حالية، أعلى سرعة، متوسط سرعة، مسافة ومدة.
- حفظ الفيديوهات داخل الجهاز باستخدام IndexedDB.
- مشاركة أو تنزيل الفيديو.
- Wake Lock لمنع قفل الشاشة أثناء التسجيل عندما يكون مدعومًا.
- واجهة عربية RTL.

## طريقة التشغيل على الآيفون
1. ارفع الملفات على استضافة HTTPS مثل GitHub Pages أو Cloudflare Pages.
2. افتح الرابط من Safari.
3. اضغط مشاركة ثم إضافة إلى الشاشة الرئيسية.
4. افتح التطبيق من الأيقونة.
5. اسمح للكاميرا والموقع.

## ملاحظات مهمة
- يجب أن يبقى التطبيق مفتوحًا أثناء التسجيل.
- التسجيل الطويل جدًا قد يستهلك مساحة وذاكرة المتصفح، لذلك استخدم مقاطع قصيرة.
- على بعض إصدارات iOS قد تختلف صيغة الفيديو المدعومة بين MP4 و WebM حسب المتصفح والنظام.
- لا تستخدم الهاتف أثناء القيادة؛ ثبته قبل الانطلاق.
